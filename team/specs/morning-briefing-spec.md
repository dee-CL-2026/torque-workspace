# Morning Briefing Spec

- Schedule: Daily, first heartbeat between 07:00–08:00 Jakarta time (GMT+7).
- Content (5–10 lines max):
  - Weather summary for Jakarta.
  - Today’s calendar (when available).
  - Urgent tasks from `tasks.md`.
  - Overnight agent activity from `activity-log.json`.
- Delivery: Send to Dee via Telegram.
- Model: Cheapest available.
- Format: Concise, plain text, 5–10 lines total.
