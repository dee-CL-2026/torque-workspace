## ‚ö†Ô∏è LESSON LEARNED: WhatsApp Config

**What I did wrong:**
```json
"whatsapp": {
  "enabled": true,  // ‚ùå WRONG - WhatsApp doesn't use this key
  "groups": ["*"]   // ‚ùå WRONG - expects object/record, not array
}
```

**Correct format:**
```json
"whatsapp": {
  "selfChatMode": true,
  "dmPolicy": "allowlist",
  "allowFrom": ["+6282112747842"],
  "groupPolicy": "allowlist"
  // groups should be an object like { "groupJid": { ... } }, not ["*"]
}
```

**Result:** Config validation failed repeatedly, caused ~18 minute restart loop. Dee had to manually fix.

**For next time:** Check docs/channels/whatsapp.md before editing WhatsApp config. Don't assume it mirrors Telegram structure.
